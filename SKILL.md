# 网页搜索技能
调用内置浏览器执行网络搜索，支持单次多条查询，返回网页原始搜索结果及AI归纳摘要。

## 功能说明
- 单次最多同时支持3条独立查询语句并行搜索；
- 返回内容包含：网页搜索原文摘要 + AI整合回复；
- AI摘要仅作参考，关键信息请以原始网页内容为准；

## 使用方法
- **命令模板**: `curl -G "http://127.0.0.1:10003/webSearch" --data-urlencode query=query1  --data-urlencode query=query2 --data-urlencode query=query3`
- **参数解释**:
  - `query`: 搜索查询语句，最多支持3条
- **命令示例**: `curl -G "http://127.0.0.1:10003/webSearch" --data-urlencode query=海绵宝宝的朋友是谁  --data-urlencode query=肯德基最好喝的咖啡`

## 使用场景
### 何时使用
- 需要实时时效性信息（新闻、价格、新规、实时事件等）；
- 自身知识库无相关内容、信息过时，需要联网补充佐证；
- 需要查询专业资料、教程、官方文档、事实性考证。

### 何时不使用
- 自身已有准确、完备、未过时的知识，可直接作答无需联网；
- 纯创作、闲聊、逻辑推理、主观观点类对话，无需外部事实信息；
- 模糊无明确检索关键词、无法生成有效查询语句时。
